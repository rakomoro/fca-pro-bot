# 🤖 FCA Pro Bot (Manus Edition)

هذا بوت احترافي ومتطور لمنصة فيسبوك ماسنجر، تم بناؤه باستخدام مكتبة `ws3-fca` مع ميزات ترفيهية وألعاب ونظام ذكاء اصطناعي وقاعدة بيانات متكاملة.

## 🚀 الميزات الرئيسية:
- **نظام أوامر متقدم**: معالج أوامر ذكي يدعم الردود والتفاعلات.
- **قاعدة بيانات SQLite**: لحفظ بيانات المستخدمين، الأموال، والمستويات.
- **ألعاب جماعية وتنافسية**: (المغارة، الحرب، فري فاير، الياكوزا).
- **ذكاء اصطناعي**: مدمج للرد على استفسارات المستخدمين.
- **نظام اقتصادي**: عملات، مكافآت يومية، ورصيد.
- **سهولة التخصيص**: هيكلية منظمة للمجلدات والملفات.

## 🛠️ المتطلبات:
- Node.js (الإصدار 16 أو أعلى)
- ملف `appstate.json` (لتسجيل الدخول)

## 📦 التثبيت:
1. قم بتحميل السورس أو عمل `git clone`.
2. افتح المجلد في Terminal.
3. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```
4. ضع ملف `appstate.json` في المجلد الرئيسي.
5. قم بتشغيل البوت:
   ```bash
   npm start
   ```

## 📜 قائمة الأوامر:
- `/help`: عرض قائمة الأوامر.
- `/balance`: التحقق من رصيدك.
- `/daily`: الحصول على المكافأة اليومية.
- `/ai [سؤال]`: التحدث مع الذكاء الاصطناعي.
- `/cave`: لعبة المغارة.
- `/war`: لعبة الحرب.
- `/freefire`: لعبة فري فاير.
- `/yakuza`: لعبة الياكوزا.
